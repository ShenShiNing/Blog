## Youtube Clone项目

这是一个类似Youtube的视频分享平台，允许用户上传、观看和分享视频。

### 技术栈

该项目使用了现代前端和后端技术:

- **前端**: React, Next.js, TypeScript, shadcn/ui
- **样式**: Tailwind CSS
- **API**: tRPC
- **数据库**: Drizzle ORM
- **认证**: Clerk Authentication
- **文件上传**: Uploadthing
- **队列处理**: Upstash

### 主要功能

1. **视频上传与管理**

   - 支持多种格式视频上传
   - 自动生成缩略图
   - 视频元数据编辑

2. **用户交互**

   - 点赞和评论系统
   - 分享功能
   - 用户频道订阅

3. **搜索与发现**

   - 全文搜索
   - 分类浏览
   - 推荐算法

4. **响应式设计**
   - 移动端完美适配
   - 深色/浅色模式
   - 无障碍功能支持

### 技术亮点

项目采用了现代化的技术架构，包括服务器组件和客户端组件的合理分离，有效提升了应用性能和用户体验。通过tRPC实现了类型安全的API调用，减少了前后端沟通成本。
